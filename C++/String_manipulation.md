# String Manipulation  

## Purpose of document  
이 문서는 종종 사용되는 C++ 의 다양한 문자열 연산 및 조작을 저장하여  
필요한 경우 참고를 목적으로 한다.

### Basic member functions of String class
1. `.append()`  
**+** 기호와 같은 기능을 수행한다.  
위 함수는 대상 스트링 변수에 문자를 추가한다.  
추가시에는 변수의 뒤에 문자를 삽입한다.  

2. `clear()`
대상 스트링 변수의 모든 내용을 제거한다.  
제거 후 값은 NULL이 아닌 비어있는 값이 된다.
